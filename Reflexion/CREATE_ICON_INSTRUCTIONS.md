# Create Icon: assets/icon.png

## Requirements
- Size: 1024×1024 pixels
- Format: PNG
- Design: Neon cyan→magenta radial gradient, center "R" glyph (white, Orbitron style)

## Quick Creation Options

### Option 1: Online Tool
1. Go to https://www.canva.com or https://www.figma.com
2. Create 1024x1024 canvas
3. Add radial gradient (cyan #00F5FF → magenta #FF00FF)
4. Add white "R" text, Orbitron Bold, centered, ~600px font size
5. Export as PNG → save to `assets/icon.png`

### Option 2: Photoshop/GIMP
1. New image: 1024x1024px
2. Gradient tool: Radial, cyan (#00F5FF) to magenta (#FF00FF)
3. Text tool: "R", white, Orbitron Bold, ~600pt, centered
4. Outer glow effect (optional)
5. Export PNG → `assets/icon.png`

### Option 3: Placeholder (Quick)
Use any 1024x1024 PNG as placeholder until you create the proper design icon.

**File must exist at**: `./assets/icon.png`
